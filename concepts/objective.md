# Objective

An objective is a declared intended outcome that authorizes and bounds autonomous execution.

## What an objective is
- a stable identifier and description of intended work
- a reference point for attributing actions
- a unit that can be reviewed, controlled, and reported

## What an objective is not
- not an implementation plan
- not a hidden model prompt
- not an implicit assumption inferred from behavior

## Why objectives matter
Objectives provide the anchor for attribution, control, trace analysis, and conformance evidence.
