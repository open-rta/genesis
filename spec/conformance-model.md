# Conformance Model

## Claiming Open RTA alignment
A runtime may claim Open RTA alignment when it can expose, from its own project root, artifacts that satisfy the laws and corresponding schema-backed requirements.

## Minimum exposed artifact classes
- objective declaration
- authority declaration
- trace events
- oversight record
- control record
- replay or reconstruction record
- conformance report

## Role of evidence
Conformance is evidence-driven. Assertions are insufficient without inspectable artifacts.

## Scope boundary
This model defines minimum conditions for alignment. It does not define certification bureaucracy or accrediting institutions.
