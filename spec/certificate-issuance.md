# Certificate Issuance

## Core rule

Validator pass != certificate issuance.

## Role of validator output

The Genesis validator provides pre-certification readiness checks:
- manifest correctness
- evidence discoverability
- schema conformance for known artifacts
- basic laws/level prerequisite consistency

This output is intended to reduce review friction, not replace review judgment.

## Formal compliance and certification

Foundation/manual review may still be required for formal compliance decisions and any certification status, especially L4.
